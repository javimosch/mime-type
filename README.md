# mime-type
Detect MIME types from filenames. Written in Go.
```bash
curl -LO https://github.com/javimosch/mime-type/releases/latest/download/mime-type-linux-amd64
chmod +x mime-type-linux-amd64
sudo mv mime-type-linux-amd64 /usr/local/bin/mime-type
```
Usage: `mime-type image.png` → `image/png`

---

This tool is a plugin for [supercli](https://github.com/javimosch/supercli) — an AI-friendly, config-driven dynamic CLI platform.
